# Sample draft — automatic citations demo

Genre analysis models a text type as an ordered set of communicative moves [@swales1990]. Writers also position themselves interpersonally: @hyland2005 describes metadiscourse as the resources through which authors interact with readers. When such qualities are scored on ordered category scales, the Rating Scale Model provides the measurement foundation [@andrich1978].

Multiple sources can share one citation [@swales1990; @hyland2005], and page locators work too [@hyland2005, p. 37].
